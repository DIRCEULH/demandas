
Objetivo do Projeto

Este projeto tem como objetivo coletar e analisar dados de todos os produtos vendidos, a fim de entender o comportamento da demanda. A partir dessa análise, busca-se projetar a quantidade ideal de itens a serem comprados para reposição, considerando uma previsão para os próximos 4 meses.

🔍 Descrição

A solução consiste em um sistema web integrado ao ERP da Senior, responsável por:

Coletar dados históricos de vendas diretamente do ERP;
Analisar a demanda por produto;
Identificar padrões de consumo;
Projetar a demanda futura com base nos dados analisados;
Gerar insights para reposição de estoque.


⚙️ Integração

O sistema se integra ao ERP da Senior para garantir:

Sincronização automática de dados de vendas;
Atualização contínua das informações;
Confiabilidade e consistência dos dados utilizados na análise.


🎯 Resultado Esperado

Com essas informações, o projeto permite:

Melhorar o planejamento de estoque;
Evitar falta ou excesso de produtos;
Otimizar o processo de reposição;
Apoiar a tomada de decisão com base em dados.
